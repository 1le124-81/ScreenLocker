# ScreenLocker
simple code that shows a popup at a chosen time in order to help you not lose track of time.

When the code is started an input time can be set in a window by the user, and then when the inputted time has been reached, it will show a popup and also hit the space key. i added this because i watch youtube a lot while procrasinating.

time is in the format HH:MM. Theoretically it works with H:MM too and has checks for the correct time format as to not break.

if the code has been running for more than 30 minutes, it wil automatically time itself out and end the code. this is to prevent memory drain.

this program uses datetime and time for the computer time, sys for exiting the code, and tkinter for the popups.
